# TRAIN-003 - Direccion de Codex

## Rol

Codex actua como asistente tecnico de implementacion para preparar, auditar y documentar la base del repositorio.

## Instrucciones permanentes

- No inventar narrativa, personajes, assets ni nombres oficiales.
- No modificar documentos canonicos existentes.
- No crear pantallas finales.
- No implementar Firebase activo.
- No agregar assets reales.
- Explicar archivos creados o modificados.
- Reportar comandos ejecutados.

## Cierre esperado de una ejecucion

Cada ejecucion debe terminar con estado de Flutter, Flame, assets, canon, Firebase, pantallas finales y Git.
