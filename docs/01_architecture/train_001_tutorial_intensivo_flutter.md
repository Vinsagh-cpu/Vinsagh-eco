# TRAIN-001 - Tutorial intensivo Flutter

## Objetivo

Dejar una guia operativa minima para que el equipo pueda leer, ejecutar y modificar una app Flutter sin convertir PACKAGE-001 en una fase de producto final.

## Ruta de aprendizaje

- Entender `pubspec.yaml`, `lib/main.dart` y el ciclo `flutter run`.
- Distinguir widgets tecnicos temporales de pantallas finales.
- Practicar estructura por carpetas antes de crear UI definitiva.
- Revisar errores de consola antes de agregar dependencias.
- Mantener cada cambio pequeno, nombrado y revisable.

## Regla PACKAGE-001

Todo ejemplo debe ser tecnico. No se crean pantallas finales, assets reales ni narrativa nueva.
